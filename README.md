### FEATURES:-
* Videocall
* Sharable Whiteboard
* Realtime Chat
* Notes (Downloadable in pdf format)
* Mute/Unmute
* Authentication
* Neat Ui
## Steps To Run In Your Machine
#### cd video-chat
#### npm i
#### npm run server
### Open the second terminal
#### cd client
#### npm i
#### npm run start
